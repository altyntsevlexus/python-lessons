🏠 Намалюй будинок з навколишнім пейзажем

Напиши програму на Python Turtle, яка малює будинок з парканом. Програма повинна запускатись без помилок і показувати результат на екрані.

Сцена має містити: зелену землю, стіни будинку, дах, двері, вікно та паркан. Для паркану обов'язково використай цикл for. Усі фігури мають бути залиті кольором.

Нагадування — Послідовність малювання
01
t.up()
підняти перо — рух без лінії
02
t.goto(x, y)
перейти до початкової точки
03
t.down()
опустити перо — малюємо
04
t.fillcolor()
задати колір заливки
05
t.begin_fill()
почати залиту фігуру
06
малюю фігуру
forward, left, circle…
07
t.end_fill()
залити і завершити
t.up() # підняти перо — переміщення без лінії t.goto(x, y) # перейти до початкової точки фігури t.down() # опустити перо — починаємо малювати t.fillcolor("#КОЛІР") # колір заливки — до begin_fill! t.begin_fill() # ← старт фігури # ... малюю фігуру ... t.end_fill() # ← заливаємо і завершуємо
