🚀 Намалюй космічний пейзаж

Напиши програму на Python Turtle, яка малює космічний пейзаж. Програма повинна запускатись без помилок і показувати результат на екрані.

Обов'язково: зірки — використай функцію def star(), цикл for, розгалуження if / elif / else та random.
